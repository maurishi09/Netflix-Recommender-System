# Netflix Movie Recommendation System

A personalized movie recommender built using collaborative filtering with SVD, based on user ratings.

## What this project does

Predicts which movies a user is likely to enjoy based on patterns in how similar users have rated films, rather than relying on genre tags or movie metadata alone.

## Approach

- Preprocessed and handled missing data in the ratings dataset
- Built a collaborative filtering model using SVD (Singular Value Decomposition)
- Evaluated performance using RMSE, MAE, Precision, and Recall

## Stack

Python · Pandas · NumPy · Scikit-learn · Surprise · Matplotlib · Seaborn
